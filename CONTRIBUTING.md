# Как внести вклад

Спасибо за интерес к проекту! Вот несколько способов помочь:

## Баг-репорты

Создайте Issue с описанием:
- Версия Ubuntu
- Полный вывод скрипта (если есть ошибка)
- Шаги для воспроизведения

## Pull Requests

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b fix/описание`
3. Внесите изменения
4. Убедитесь, что скрипт проходит `shellcheck install_mtproxy.sh`
5. Создайте Pull Request

## Стиль кода

- Bash с `set -euo pipefail`
- Комментарии на русском
- Функции для логирования: `info()`, `ok()`, `warn()`, `fail()`
